Types of Pipes

There are different types of Pipes in Angular.

Async Pipe

One-way data binding basically work component to view.

Date Pipe

Two-way data binding basically work component to view and view to component.

Slice Pipe

One-way data binding basically work component to view.

Lower Case Pipe

Two-way data binding basically work component to view and view to component.

Json Pipe

One-way data binding basically work component to view.

Currency Pipe

Two-way data binding basically work component to view and view to component.

Title case Pipe

Two-way data binding basically work component to view and view to component.
