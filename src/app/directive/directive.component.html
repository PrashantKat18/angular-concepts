Directives Definition/Types

Basically Directive are elements which change the structure ,behavior or appereance of the Dom elements.

Types Of Directive

There are four types of directives in Angular.

Structural directives Attribute directives Components Directives Custom Directive Structural Directives Example

Structural directive change the structure of the DOM by adding or removing elements.

It's denote by *Sign.

Like - *ngIf ,*ngFor, *ngSwitch

 Attribute Directives Example

Attribute directiv basically perform the apperence of DOM elements or Components.

Like- ngStyle,ngClass

Components Directives Example

Component directives are based on component. Actually, each component can be used as directive. Component provides @Input and @Output decorator to send and receive information between parent and child components.

Like- @Input

Custom Directives Example

Angular provides option to extend the angular directive with user defined directives and it is called Custom directives.

Command to create custom directive -> ng generate directive customstyle
